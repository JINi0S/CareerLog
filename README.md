## CareerLog 커리어 로그
CareerLog는 자기소개서를 체계적으로 관리하고 작성할 수 있도록 도와주는 앱입니다.

[앱스토어](https://apps.apple.com/kr/app/%EC%BB%A4%EB%A6%AC%EC%96%B4-%EB%A1%9C%EA%B7%B8/id6748353450?mt=12)

### 💻 개발 기간

2025년 6월 ~ 진행 중

</br>

### ✏️ 주요 기능

- 자기소개서 질문/답변 항목별 등록 및 수정
- 항목 순서 변경 및 삭제
- 질문 유형에 따른 태그 기능
- 다크모드 대응

</br>
